# 📄 Templates

|----------|------|-------------|
| [`github-pages-index.html`](github-pages-index.html) | Docsify | `index.html` for Docsify + GitHub Pages. VS Code dark theme, search, sidebar, inline code fix. Rename to `index.html`. |
